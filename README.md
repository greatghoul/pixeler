## Pixeler
一个用于设计像素图片（拼豆）的应用。用HTML5 canvas开发。  
A website to design pixel picture, based on HTML5 canvas.

## Version

### V0.2 20150916
能更换拼豆颜色。
Change the color.

### V0.1 20150911
实现绘制、清除拼豆。
Draw and clear pixels.
